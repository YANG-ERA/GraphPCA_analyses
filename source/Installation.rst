Installation
============
GraphPCA package is mainly based on python.

Software dependencies
---------------------
.. code-block:: python

   numpy
   pandas
   matplotlib
   scipy
   scikit-learn
   networkx
   scanpy
   squidpy



installation
---------------------
Install GraphPCA via PyPI by using:

.. code-block:: shell

   pip install st-graphpca
